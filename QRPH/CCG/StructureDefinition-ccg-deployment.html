Resource Profile: CCG Deployment Package (IG)

Official URL: https://profiles.ihe.net/QRPH/CCG/StructureDefinition/ccg-deployment Version: 1.0.0-comment
Active as of 2025-05-20 Computable Name: CCG_Deployment_Package

The deployment model is that each unique CCG Folder and its contents will be contained in a FHIR package.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
