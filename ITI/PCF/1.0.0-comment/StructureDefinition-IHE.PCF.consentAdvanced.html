Resource Profile: IHE PCF Explicit Advanced Consent

Official URL: https://profiles.ihe.net/ITI/PCF/StructureDefinition/IHE.PCF.consentAdvanced Version: 1.0.0-comment
Active as of 2023-05-03 Computable Name: AdvancedConsent

Explicit Advanced Consent

  • all elements allowed by Basic and Intermediate are allowed here, plus the following
  • securityLabel indicates sensitivity or confidentiality tags on data
    • Only codes from Avanced Security Tag ValueSet

Usage:

  • Examples for this Resource Profile: Consent/ex-consent-advanced-normal-break-glass-restricted, Consent/ex-consent-advanced-normal-focused-psy-or-sex, Consent/ex-consent-advanced-normal-focused-psy, Consent/ex-consent-advanced-normal-focused-restricted... Show 3 more, Consent/ex-consent-advanced-normal-not-restricted, Consent/ex-consent-advanced-normal-restricted and Consent/ex-consent-advanced-normal
