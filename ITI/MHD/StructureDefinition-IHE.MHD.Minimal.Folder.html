Resource Profile: MHD Folder Minimal

Official URL: https://profiles.ihe.net/ITI/MHD/StructureDefinition/IHE.MHD.Minimal.Folder Version: 4.2.0
Active as of 2022-12-07 Computable Name: Folder

A profile on the List resource for MHD use as a Folder with minimal metadata constraints.

  • MHD is based on the IHE Document Sharing model,
  • the 3:4.1 Abstract Metadata Model, and
  • the use defined here is FHIR List for use as a Folder implementation of the
  • ebRIM implementation at 3:4.2.3.4 Folder Attributes.
  • with use-cases and constraints found in 3:4.3 Additional Document Sharing Requirements

Usage:

  • Derived from this Resource Profile: MHD Folder Comprehensive
  • Use this Resource Profile: MHD Comprehensive Provide Document Bundle, MHD Find Document Lists Response Message, MHD Minimal Provide Document Bundle, MHD UnContained Comprehensive Provide Document Bundle
  • Examples for this Resource Profile: List/ex-minimalFolder
