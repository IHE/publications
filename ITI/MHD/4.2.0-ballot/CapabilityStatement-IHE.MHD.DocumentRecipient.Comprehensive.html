IHE ITI Mobile access to Health Documents (MHD) - Document Recipient - with Comprehensive declared Option.

This is the Requirements CapabilityStatement for a Document Recipient with Comprehensive Metadata Option declared Option. This actor must use Comprehensive Metadata Option. Note that XDS on FHIR Option requires the Comprehensive Metadata Option.

The Document Source uses the Provide Document Bundle ITI-65 to communicate metadata and documents to the Document Recipient.
