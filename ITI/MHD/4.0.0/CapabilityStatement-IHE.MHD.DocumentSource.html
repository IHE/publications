CapabilityStatement: Document Source (publisher)

Raw OpenAPI-Swagger Definition file | Download

This is the Requirements CapabilityStatement for a Document Source with no declared Options, or all declared Options. This Actor is allowed to use Minimal Metadata, Comprehensive Metadata Option, XDS on FHIR Option, or UnContained Comprehensive Metadata.

The Document Source uses the Provide Document Bundle ITI-65 to communicate metadata and documents to the Document Recipient.
