Resource Profile: FAIS EnrollmentResponse Profile

Official URL: https://profiles.ihe.net/ITI/FAIS/StructureDefinition/IHE.FAIS.EnrollmentResponse Version: 1.0.0
Active as of 2024-11-13 Computable Name: FAISEnrollmentResponse

The FAIS EnrollmentResponse profile to be used as the response to the FAIS EnrollmentRequest $submit

Usage:

  • Use this Resource Profile: FAIS EnrollmentResponse Bundle Profile
  • Examples for this Resource Profile: EnrollmentResponse/ex-enrollment-resp-1 and EnrollmentResponse/ex-enrollment-resp
