Extension: AuditEvent.agent other identifiers

Official URL: https://profiles.ihe.net/ITI/BALP/StructureDefinition/ihe-otherId Version: 1.1.4
Active as of 2025-10-31 Computable Name: OtherId

Carries other identifiers are known for an agent.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: AuditEvent.agent

Usage info

Usages:

  • Use this Extension: Basic AuditEvent pattern for when an activity was authorized by an SAML access token Comprehensive
  • Examples for this Extension: AuditEvent/ex-auditPoke-SAML-Comp and AuditEvent/ex-auditPoke-SAML-QDI-Comp
