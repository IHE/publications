Structures: Resource Profiles

These define constraints on FHIR resources that need to be complied with by conformant implementations

6.6.107 IHE ACDC Questionnaire Profile Defines constraints and extensions on the Questionnaire Resource for questionnaires used with the IHE ACDC Profile
6.6.108 IHE ACDC QuestionnaireResponse Profile Defines constraints and extensions on the QuestionnaireResponse Resource for responses used with the IHE ACDC Profile
IHE ACDC Query Artifact AuditEvent Profile Defines constraints and extensions on the AuditEvent Resource for audit messages used with the IHE ACDC Profile [PCC-71] Request Assessment Transaction

Structures: Extension Definitions

These define constraints on FHIR data types that need to be complied with by conformant implementations

6.7.3 IHE ACDC Launch URL Extension Defines the Launch URL Extension used on Questionnaire Resources used by the IHE ACDC Profile
6.7.4 IHE ACDC Contained Questionnaire Reference Extension Defines the Contained Questionnaire Reference URL Extension used on QuestionnaireResponse Resources used by the IHE ACDC Profile
