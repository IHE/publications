Logical Model: Dosaging (model)

Official URL: https://profiles.ihe.net/PHARM/MPD/StructureDefinition/DosagingInformation Version: 1.0.0-preview
Active as of 2025-05-08 Computable Name: DosagingInformation

Copyright/Legal: IHE http://www.ihe.net/Governance/#Intellectual_Property

Logical model for usage instructions for administring the requested product. Based on FHIR Dosage complex data type.

Usage:

  • Refer to this Logical Model Profile: Medication dispense (model), Medication order (model) and Medication order - multiline (model)
