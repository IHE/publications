introductionThe Medication Prescription and Dispense (MPD) profile defines profiles and actors for interoperable systems handling medication ordering and dispensing. Based on the previous work done in the IHE Pharmacy Technical Framework, the MPD profile uses FHIR and is designed to support the exchange of medication orders and dispenses between healthcare providers, pharmacies, and other stakeholders in the medication management process. The MPD profile supports several use cases - some examples: - ordering (from draft or proposed orders to actionable prescriptions); - order grouping (i.e. multiline prescriptions); - instance ordering (i.e. dispense requests); - dispense (including partial dispense, declined dispense).
